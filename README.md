# TileFlow -- конфигуратор оконных менеджеров
---
## Особенности
- Написан на Gtk библиотеке
- Имеет tui консоль(в будующем заменится на команду с аргументами)
- список не закончен

## Цели
- Написать Hyprland парсер
- Реализовать работу со строками
- Дополнения(поиск, удаление, перемещение)
- Локализация значений
- Привод кода парсера к кроссплатформенности

---
## Установка
- Скачайте пакеты pygobject/gtk для вашего дистрибутива
```sh
git clone https://github.com/sophrosha/tileflow
cd tileflow
python3 main.py
```
